Part 1: Installing Python & Anaconda

(8 minutes)

Why Anaconda?

Python Alone: - Need to install packages one by one - Managing dependencies is painful - Different projects need different versions - Environment conflicts Anaconda: ‚úì Python + 250+ data science packages ‚úì Includes Jupyter Notebooks ‚úì Package management (conda) ‚úì Environment management ‚úì Works on Windows, Mac, Linux ‚úì One installation = Everything you need

Think of it like this:

  • Just Python = Buying a car with no accessories
  • Anaconda = Buying a fully loaded car with GPS, premium sound, everything

For Windows Users:

Step-by-Step Installation

Step 1: Download ‚Üí Go to: https://www.anaconda.com/download ‚Üí Click "Download" for Windows ‚Üí Choose 64-bit installer (most computers) ‚Üí File is ~500MB, takes 2-5 minutes to download Step 2: Run Installer ‚Üí Double-click the .exe file ‚Üí Click "Next" through the screens ‚Üí IMPORTANT: Check "Add Anaconda to PATH" (makes life easier) ‚Üí Choose "Install for: Just Me" (recommended) ‚Üí Installation takes 5-10 minutes Step 3: Verify Installation ‚Üí Open Command Prompt (search "cmd" in Start menu) ‚Üí Type: conda --version ‚Üí Should see: conda 23.x.x (some version number) ‚Üí Type: python --version ‚Üí Should see: Python 3.11.x or similar

Common Windows Issues

Issue: "conda not recognized"
Solution: Need to add to PATH manually

  1. Search "Environment Variables" in Start
  2. Edit "Path" variable
  3. Add: C:\Users\[YourName]\anaconda3\Scripts
  4. Restart Command Prompt

For Mac Users:

macOS Installation

Step 1: Download ‚Üí Go to: https://www.anaconda.com/download ‚Üí Click "Download" for macOS ‚Üí Choose the right chip: - M1/M2/M3 Mac: Apple Silicon - Older Mac: Intel ‚Üí File is ~500MB Step 2: Run Installer ‚Üí Double-click the .pkg file ‚Üí Click "Continue" through screens ‚Üí Install in default location ‚Üí Takes 5-10 minutes Step 3: Verify Installation ‚Üí Open Terminal (Cmd+Space, type "Terminal") ‚Üí Type: conda --version ‚Üí Should see: conda 23.x.x ‚Üí Type: python --version ‚Üí Should see: Python 3.11.x

For Linux Users:

Linux Installation

Step 1: Download wget https://repo.anaconda.com/archive/Anaconda3-2024.02-1-Linux-x86_64.sh Step 2: Install bash Anaconda3-2024.02-1-Linux-x86_64.sh ‚Üí Follow prompts, accept license ‚Üí Install in default location: ~/anaconda3 ‚Üí Yes to "conda init" Step 3: Verify ‚Üí Close and reopen terminal ‚Üí conda --version ‚Üí python --version

Part 2: Jupyter Notebooks

(5 minutes)

What is Jupyter Notebook?

Interactive coding environment: - Write code in cells - Run cells individually - See output immediately - Mix code, text, and visualizations - Perfect for learning and experimentation Why Data Scientists Love It: ‚úì Immediate feedback ‚úì Document your thinking ‚úì Share work easily ‚úì Visualize results inline ‚úì Experiment without risk

Think of it like a lab notebook:

  • Traditional coding: Write entire program ‚Üí Run ‚Üí See all results
  • Jupyter: Write small piece ‚Üí Run ‚Üí See result ‚Üí Write next piece

Launching Jupyter Notebook

Method 1: Anaconda Navigator (Easy, Visual) Step 1: Open Anaconda Navigator ‚Üí Windows: Search "Anaconda Navigator" in Start ‚Üí Mac: Applications ‚Üí Anaconda Navigator Step 2: Launch Jupyter ‚Üí Find "Jupyter Notebook" tile ‚Üí Click "Launch" ‚Üí Browser opens automatically ‚Üí You'll see your file system Method 2: Command Line (Faster) Step 1: Open Terminal/Command Prompt Step 2: Type command $ jupyter notebook Step 3: Browser opens automatically ‚Üí Shows: http://localhost:8888/tree ‚Üí You see your home directory

Creating Your First Notebook

Step 1: Click "New" (top right) Step 2: Select "Python 3" Step 3: You now have a blank notebook! Step 4: Rename it (click "Untitled" at top)

Your First Code

# Cell 1: Basic Python print("Hello, AI World!") print("2 + 2 =", 2 + 2) # Press Shift+Enter to run

Keyboard Shortcuts (Most Useful)

Command Mode (press Esc):

  • A: Add cell above
  • B: Add cell below
  • DD: Delete cell
  • M: Change to Markdown
  • Y: Change to Code

Edit Mode (press Enter):

  • Shift+Enter: Run cell, move to next
  • Ctrl+Enter: Run cell, stay in cell
  • Alt+Enter: Run cell, insert below

(3 minutes)

The AI/ML Python Stack

NumPy: Numerical computing, arrays Pandas: Data manipulation, DataFrames Matplotlib: Data visualization, plots Scikit-learn: Classical ML algorithms PyTorch: Deep Learning (Week 3+) Good News: Anaconda includes most of these!

If all these work, you're set! ‚úì

If Something's Missing

Part 4: Google Colab (Cloud Alternative)

(4 minutes)

Google Colab: Jupyter in the Cloud

Advantages: ‚úì No installation needed ‚úì Access from anywhere ‚úì Free GPU access ‚úì Preinstalled libraries ‚úì Easy sharing ‚úì Saves to Google Drive When to Use: - Need GPU for deep learning - On a different computer - Installation problems - Quick experiments - Collaborating with others

"Sometimes you can't or don't want to use your local computer:

  • Traveling with tablet
  • Installation issues
  • Need more power
  • Want to collaborate

That's where Google Colab shines!

Getting Started with Colab

Step 1: Open browser ‚Üí Go to: https://colab.research.google.com Step 2: Sign in ‚Üí Use Google account ‚Üí Don't have one? Create free account Step 3: Create notebook ‚Üí Click "New Notebook" ‚Üí Or: File ‚Üí New Notebook

Key Differences: Colab vs Local Jupyter

Colab: - Runs on Google servers - Files save to Google Drive - Can enable GPU/TPU - Already has libraries installed - Free tier has time limits Local Jupyter: - Runs on your computer - Files save locally - Uses your CPU/GPU - You install libraries - No time limits

Enabling GPU (For Deep Learning)

Step 1: Runtime ‚Üí Change runtime type Step 2: Hardware accelerator: GPU Step 3: Save Now you have free GPU! üöÄ

Testing Colab

Run the same code:

# Already installed in Colab! import numpy as np import pandas as pd import matplotlib.pyplot as plt import torch print(f"NumPy: {np.__version__}") print(f"Pandas: {pd.__version__}") print(f"PyTorch: {torch.__version__}") # Check if GPU available if torch.cuda.is_available(): print("‚úì GPU is available!") else: print("Using CPU (enable GPU in Runtime settings)")

Best Practice

I recommend:

  • Primary: Use local Jupyter for daily work
  • Backup: Use Colab when you need GPU or are away from computer
  • Collaboration: Share Colab links with teammates

All our course materials will work on both!

Common Issues & Fixes

Problem: "ModuleNotFoundError: No module named 'numpy'"
Solution: $ conda install numpy or restart Jupyter after installation

Problem: Plot doesn't show
Solution: Add this to first cell: %matplotlib inline

Problem: Jupyter won't start
Solution:

  1. Check if already running (close other instances)
  2. Try: jupyter notebook --port 8889 (different port)
  3. Reinstall: conda install jupyter
