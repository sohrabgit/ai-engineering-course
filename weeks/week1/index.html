Week 1

AI Fundamentals & Python for AI

Understand what AI, Machine Learning, and Deep Learning are. Set up a complete AI development environment and master essential Python libraries (NumPy, Pandas, Matplotlib). Build your first simple ML model.

2 Sessions
210 Total Minutes
4-6 Hours This Week

Learning Objectives

By the end of this week, you will be able to:

  • Understand what AI, Machine Learning, and Deep Learning are
  • Set up a complete AI development environment
  • Master essential Python libraries for AI (NumPy, Pandas, Matplotlib)
  • Build your first simple ML model

Week 1 Project: Build Your First ML Model

Build a simple linear regression model to predict house prices based on features like size, location, and number of bedrooms.

Project Overview: House Price Predictor

Data Exploration

Load and explore the housing dataset, visualize relationships between features and price

Data Preparation

Clean and prepare the data, split into training and testing sets

Model Building

Build a linear regression model using scikit-learn and evaluate performance

Predictions

Make predictions on new data and create Jupyter notebook with complete analysis

Why This Project Matters

This hands-on project introduces you to the complete machine learning workflow: data exploration, preprocessing, model training, and evaluation. You'll work with a real dataset and build something that actually works!
