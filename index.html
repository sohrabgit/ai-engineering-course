6-Week Curriculum

Comprehensive curriculum designed to take you from AI fundamentals to building intelligent systems.

Week 1: AI Fundamentals & Machine Learning Basics

Learning Objectives

  • Define artificial intelligence in simple terms
  • Distinguish between different types of AI
  • Recognize AI applications in everyday life
  • Understand the current state and future potential of AI
Session 1.1: What is Artificial Intelligence? 20 min
  • Defining AI and its core components
  • Three levels of AI: Narrow, General, and Super AI
  • Real-world AI applications across industries
  • Current capabilities and limitations

Hands-on Activity: AI recognition exercise with real-world examples

Session 1.2: Machine Learning Fundamentals 90 min
  • Introduction to Machine Learning concepts
  • Supervised vs Unsupervised vs Reinforcement Learning
  • Data preprocessing and feature engineering
  • Model training and evaluation basics

Hands-on Activity: Build your first ML model with Python and scikit-learn

Week 1 Project: AI Landscape Analysis

Research and present on AI applications in your industry of interest.

  • Identify 3 AI use cases in chosen industry
  • Analyze one use case in detail
  • Create presentation summarizing findings
  • Set up Python development environment

Week 2: Data Science & Python for AI

Learning Objectives

  • Master Python fundamentals for data science
  • Work with NumPy, Pandas, and Matplotlib
  • Understand data preprocessing techniques
  • Perform exploratory data analysis
Session 2.1: Python for Data Science 90 min
  • Python essentials: lists, dictionaries, functions
  • NumPy for numerical computing
  • Pandas for data manipulation
  • Data loading and basic operations

Hands-on Activity: Data manipulation exercises with real datasets

Session 2.2: Exploratory Data Analysis 90 min
  • Data visualization with Matplotlib and Seaborn
  • Statistical summaries and distributions
  • Identifying patterns and outliers
  • Data quality assessment

Hands-on Activity: Complete EDA on a business dataset

Week 2 Project: Data Analysis Portfolio

Complete comprehensive analysis of a real-world dataset.

  • Clean and preprocess raw dataset
  • Create visualizations showing key insights
  • Write summary of findings and recommendations
  • Document code in Jupyter notebook

Week 3: Supervised Learning Algorithms

Learning Objectives

  • Implement classification and regression algorithms
  • Understand model evaluation metrics
  • Apply cross-validation and hyperparameter tuning
  • Build end-to-end ML pipelines
Session 3.1: Classification Algorithms 90 min
  • Linear and Logistic Regression
  • Decision Trees and Random Forests
  • Support Vector Machines
  • Model evaluation and selection

Hands-on Activity: Build classification models for customer churn prediction

Session 3.2: Model Optimization & Deployment Prep 90 min
  • Feature selection and engineering
  • Hyperparameter optimization
  • Cross-validation strategies
  • Model persistence and versioning

Hands-on Activity: Optimize models and prepare for deployment

Week 3 Project: Predictive Model Build

Create production-ready machine learning model.

  • Train and optimize multiple algorithms
  • Compare model performance
  • Create model evaluation report
  • Prepare model for deployment

Week 4: Deep Learning & Neural Networks

Learning Objectives

  • Understand neural network fundamentals
  • Build deep learning models with TensorFlow/Keras
  • Work with different neural network architectures
  • Apply deep learning to image and text data
Session 4.1: Neural Network Fundamentals 90 min
  • Perceptrons and multilayer networks
  • Activation functions and backpropagation
  • Introduction to TensorFlow and Keras
  • Building your first neural network

Hands-on Activity: Implement neural network from scratch and with Keras

Session 4.2: Convolutional Neural Networks 90 min
  • CNN architecture and convolution operations
  • Image classification and computer vision
  • Transfer learning with pre-trained models
  • Image preprocessing and augmentation

Hands-on Activity: Build image classifier using CNNs

Week 4 Project: Computer Vision Application

Develop an image classification or object detection system.

  • Design and train CNN architecture
  • Implement transfer learning approach
  • Evaluate model on test dataset
  • Create demo application with user interface

Week 5: Natural Language Processing

Learning Objectives

  • Process and analyze text data
  • Build NLP models for classification and generation
  • Work with large language models and APIs
  • Implement sentiment analysis and text summarization
Session 5.1: Text Processing & Traditional NLP 90 min
  • Text preprocessing and tokenization
  • TF-IDF and word embeddings
  • Sentiment analysis and text classification
  • Named entity recognition

Hands-on Activity: Build sentiment analysis model for product reviews

Session 5.2: Large Language Models & APIs 90 min
  • Understanding transformer architecture
  • Working with OpenAI, Anthropic, and other APIs
  • Prompt engineering best practices
  • Text generation and completion

Hands-on Activity: Create AI-powered chatbot using LLM APIs

Week 5 Project: NLP Application

Build intelligent text processing application.

  • Implement text classification system
  • Integrate large language model API
  • Create text summarization feature
  • Build interactive web interface

Week 6: AI Deployment & Capstone Project

Learning Objectives

  • Deploy AI models to production environments
  • Build end-to-end AI applications
  • Understand MLOps and model monitoring
  • Complete comprehensive capstone project
Session 6.1: Model Deployment & MLOps 90 min
  • Deployment strategies and platforms
  • API development with Flask/FastAPI
  • Containerization with Docker
  • Model monitoring and maintenance

Hands-on Activity: Deploy model as REST API service

Session 6.2: Capstone Project Workshop 90 min
  • Project planning and architecture design
  • Integration of multiple AI components
  • User interface and experience design
  • Project presentation and documentation

Hands-on Activity: Work on capstone project with instructor support

Week 6 Capstone: Complete AI Application

Design and build comprehensive AI system combining multiple techniques.

  • Plan and design end-to-end AI application
  • Implement multiple AI components (ML, NLP, Computer Vision)
  • Deploy application with user interface
  • Present project to class with live demonstration
  • Document project architecture and learnings
